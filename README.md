

# ⚔️ 3Dmigoto-Sword-Lv6

> 🧰 一款用于 **提取并转换 3Dmigoto 格式 Mod 模型** 为 `.fmt` `.ib` `.vb` 的工具。
> 转换后可使用 **TheHerta3 插件** 导入至 **Blender** 中使用。
>
> 同时还包含许多 Mod 作者常用的辅助功能。

---

## 🪄 工具用途

* 🧩 **权重损坏的旧 Mod 修复**
  修复因游戏版本更新导致的权重变更、顶点组编号变更等问题。
  在此类修复中，需要先从 Mod 中提取模型。

* 🎓 **建模技巧学习**
  许多 Mod 拥有丰富的制作细节（布线、UV 分布等），可用于学习参考。

* 📦 **提取 Mod 中的模型**
  本工具仅提供模型提取功能，**不规定提取后模型的进一步用途**。

---

## 🚫 禁止以下用途

* ❌ **修改他人 Mod 并以自己名义发布**
  这侵犯了原作者对其 Mod 的修改权，也可能破坏社区氛围与收益。

* 💰 **提取模型后进行二次售卖**
  严重侵犯原 Mod 作者的实际经济利益。

* ⚖️ **任何侵犯 Mod 作者权益的行为**
  请尊重作者的版权声明，合理、合法地使用本工具。

---

## ⚖️ 争议性说明

本工具在社区中存在一定争议：
部分 **AGMG 社区** 的作者认为其可能影响 Mod 作者利益，因此 **请勿在 AGMG 宣传或传播**。

> 💡 初衷：本工具最初仅用于 **修复旧 Mod**，
> 随后才逐渐发展为 **3Dmigoto 格式通用提取工具**。

如果你对本工具持负面态度，
请将批评指向 **违规使用者**，而非开发者本人 🙏。

---

## 🔒 部分闭源说明

* GUI 使用 **C# + WinUI3** 开发（开源 ✅）
* 核心插件由 **C++** 编写（闭源 🔐），例如：
  `3Dmigoto-Sword-Lv5.exe`

> 📘 本工具使用 `texconv.exe`（来自 Microsoft DirectXTK 项目）
>
> GUI 遵循 **GPL3.0 开源协议**，
> 插件部分因涉及自动化逆向功能而暂不公开。

## 🧱 插件安装方法

1. 下载插件后，点击工具页面中的 **【打开 Assets 文件夹】**
2. 将插件放入该文件夹中即可启用功能 ✅
