# 3Dmigoto-Sword-Lv6

3Dmigoto-Sword-Lv6是一款用于将3Dmigoto格式的Mod中的模型提取并转换为.fmt .ib .vb格式的工具，

转换为.fmt .ib .vb格式后就可以使用TheHerta3插件导入到Blender中进行使用了。

除此之外还有一些其他的Mod作者常用的一些功能。

# 工具用途

- 权重损坏的旧Mod修复

修复由于游戏版本更新后，权重变更或者顶点组编号变更而导致的Mod损坏，此时需要先获取到Mod中的模型。

- 建模技巧学习

Mod中往往含有非常丰富的制作细节，比如布线、UV分布等等，值得学习品鉴。

- 提取Mod中的模型

仅仅只用于提取模型，并未规定可以用提取的模型做其他事情。


# 禁止以下用途

- 禁止用于修改其他人制作好的Mod并作为自己的产品公开发布

因为这打破了其他Mod作者对于自己Mod的修改垄断权，严重点可能会导致其他作者心情不佳或收益下滑。

- 禁止用于提取Mod中的模型二次售卖

因为这打破了其他Mod作者对于自己Mod中的模型的售卖权，严重侵犯了Mod作者的实际利益。

- 禁止使用此工具侵犯Mod作者的权益

一般Mod中都会附带有版权说明，请尊重Mod作者的版权说明，禁止使用此工具侵犯Mod作者的权益。

# 争议性说明

此工具历来争议性较大，社区中不同利益立场的用户对此工具持有不同的态度，

此工具也遭到AGMG社区部分Mod作者抵制，认为此工具会破坏Mod作者的利益，请勿在AGMG中宣传或传播此工具。

此工具设计之初的意义就是修复旧Mod，其次才慢慢发展为用于提取3Dmigoto格式Mod的通用工具。

如果你对此工具有负面看法，请去评价那些违规使用此工具的人，而不是尝试攻击开发者，谢谢。

# 部分闭源说明

本工具本质上是一个GUI，GUI部分使用C# WinUI3开发，在此Github进行开源，

其部分功能为调用第三方插件，插件部分使用C++编写，是闭源的，例如3Dmigoto-Sword-Lv5.exe插件。

此外本工具用到了texconv.exe作为贴图格式转换工具，来源于微软的DirectXTK项目。

此工具基于3Dmigoto，向3Dmigoto项目致敬，因此GUI部分采用开源协议GPL3.0，

但插件部分由于自动化逆向功能过于方便，在社区内争议较大，故闭源不公开，敬请谅解。

## 一、在《英雄联盟》世界观中的“诺克萨斯精神”

诺克萨斯（Noxus）是符文之地上最具争议的国家之一。它以**力量、征服和实用主义**著称，但它的精神并非单纯的“邪恶”或“暴力”，而是建立在一种独特的理念之上：

> **“力量值得尊重，而力量可以来自任何地方。”**

换句话说，诺克萨斯的精神核心是：

> **不论出身高低，只要你有实力，就能赢得地位与尊重。**

这使得诺克萨斯与其他势力（如贵族至上的德玛西亚）形成鲜明对比。
在诺克萨斯：

* 奴隶可以成为将军（如德莱厄斯）。
* 贵族若无能，则可能被推翻。
* 魔法、科技、武力、智谋，皆被视为“力量”的不同形式。

**代表人物及体现：**

* **德莱厄斯（Darius）**：从无名士兵成长为诺克萨斯之手，象征“通过实力登顶”。
* **斯维因（Swain）**：象征智慧与谋略的力量，用头脑掌控帝国。
* **卡特琳娜（Katarina）**：以刺客的精准与忠诚体现对诺克萨斯理念的绝对奉献。
* **萨勒芬妮在皮尔特沃夫的对立中也提到过诺克萨斯精神的危险性**――因为那种“以力量为尊”的文化，容易被滥用成“强者压迫弱者”。

---

## 二、诺克萨斯精神的文化隐喻（现实层面的理解）

从玩家或社会学角度看，诺克萨斯精神像是一种**极端的功绩主义（meritocracy）**。
它强调：

* **结果胜于过程**；
* **不讲背景，只讲实力**；
* **弱肉强食的竞争逻辑**。

这种精神有吸引力，也有危险性：

| 积极面         | 消极面             |
| ----------- | --------------- |
| 鼓励奋斗、打破阶级壁垒 | 可能导致冷酷、无情、缺乏同理心 |
| 尊重实力与成果     | 容易忽视弱者与公平       |
| 以效率与实际为导向   | 缺乏人性关怀与道德约束     |

所以，**诺克萨斯精神不是简单的“邪恶”或“正义”**，而是一种关于**力量、秩序与野心**的哲学。
它质问我们：

> “当力量与道德冲突时，我们究竟选择哪一方？”

> **诺克萨斯精神 = 以力量为真理的实用主义信条。**
>
> 它代表一种信念：**不论出身、背景、手段，只要你有足够的实力，就能改变世界。**
